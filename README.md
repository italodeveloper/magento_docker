# Magento Docker
Virtualização do nosso novo ambiente de Magento em um contexto Docker, disponibilizado em modo de produção e dev.
## Presentes nessa virtualização:
- Nginx Balancer.
- Nginx Server.
- Percona (MySQL).
- PHP7.2 (11 clousters).
- Magento 1.9.3 + Last Security Patch + PHP7 Support Patch.
